SAFE | Student-Athlete Financial Education

What is SAFE SAFE utilizes real financial values specific to each student, such as the actual value of their scholarship, to create an experiential learning environment that includes video lessons (w/quizzes), bill pay (tuition, housing, books, etc.) and discretionary spending (choices: Save, Give, Spend, Invest).
